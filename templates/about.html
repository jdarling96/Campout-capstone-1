Active Access’s Campground API

The Active Acccess Campground APIs, backed by Reserve America's database, provides access
to campground data for 97% of the US and Canada's national and state/provincial parks.

Technology

Campout if a fullstack web application that utilizes Flask for the back-end and HTML, CSS, and Javascript in the front-end.
Data is stored on a PostgreSQL database. Important libraries used are Bootstrap, jQuery.
SQLAlchemy, and WTForms.

How Does it Work?

Campout uses the above API to find campground data based on the users location. Campout has up to 12 filters
when site searching to narrow down exactly what site you are looking for. Users can then save sites they come
across which can be found in there account page. Campout also has a recomendation feature that takes a users saved
site and finds new sites based off of location and amenities!
