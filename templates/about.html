Active Access’s Campground API

The Active Acccess Campground APIs, backed by Reserve America's database, provides access
to campground data for 97% of the US and Canada's national and state/provincial parks.

Technology

Campout if a fullstack web application that utilizes Flask for the back-end and HTML,CSS, and Javascript in the front-end.
Important libraries used are Bootstrap and jQuery.

How Does it Work?

Campout uses the above API to find campground data based on the users location or location entrered.
Users can favorite/save sites which is stored in a database along with there user profile.
